# Interview Scheduler

## Setup

Install dependencies with `yarn install`.

## Running Webpack Development Server

```sh
yarn start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
yarn run storybook
```
